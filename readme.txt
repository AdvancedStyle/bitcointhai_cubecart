Project: Bitcoin.in.th Merchant Account
Module: Cubecart Payment Gateway
Author: dave@advancedstyle.com

Installation:
1) Create a merchant account at http://bitcoin.in.th/merchant-account/
2) Copy all contents of the /bitcointhai_cubecart/ folder to your root Cubecart directory
3) Go to CubeCart Admin -> Payment Gateways -> Sort by "Name" -> Click the Edit pen next to Bitcoin / Bitcoin.in.th
4) Click the "Status" cross (X) icon to change to tick mark
5) Click the "Default" cross (X) icon to change to tick mark
6) Enter your API ID and API Key
7) Click the "Save" button
